## 5.1 PAYE Online (XML) — Real Time Information

| | |
|---|---|
| **Type** | XML |
| **Auth** | Government Gateway credentials |
| **Status** | Live |

**Description:** XML-based API for Real Time Information (RTI) employer submissions. Supports Full Payment Submissions (FPS) for standard employee pay data and Employer Payment Summaries (EPS) for statutory reclaims and periods of inactivity. Technical packs are published annually (2026-27 specs now available).

**Hypothesised Utility:** Required for any payroll service. Companies with employees must submit RTI data to HMRC each time they pay staff. Core to payroll processing, tax deduction reporting, and National Insurance contributions.
