## 11.1 Export Health Certificates API

| | |
|---|---|
| **Type** | REST |
| **Auth** | API Key (authenticated portal) |
| **Status** | Sandbox (V1) |

**Description:** Enables traders to apply for export health certificates for agri-food goods, check the status of applications, and check for validation errors. Described as "the first step towards enabling digital integration for moving agri-food goods." Published by DEFRA on the Trade Developer Portal.

**Hypothesised Utility:** Required for businesses exporting food, animals, or plant products that need health certification. Automates the application and status-tracking process.
